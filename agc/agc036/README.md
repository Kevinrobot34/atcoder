## A - Triangle (400点)
* 数学
* `(X1, Y1) = (0, 0)`としてOK
  - この時、三角形の面積は`|X2 * Y3 - X3 * Y2| / 2`
  - よって`S = |X2 * Y3 - X3 * Y2|`となるように決めたい。
* `(X2, Y2) = (a, 0)`とかしてしまうと、`S=|a * Y3|`となってしまい、`S`が大きな素数の時にダメ
* `(X2, Y2) = (10**9, 1)`とすると、`S=|10**9 * Y3 - X3|`となり、よしなに決められる
