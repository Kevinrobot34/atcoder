## E - Red Scarf (500点)
* keywords
  - Xor
* 解法
  - Xorは結合則があり、 `a xor a = 0`で、`0`が単位元であることがポイント
  - 上記を把握していれば数列a全てのxorを取り、それと`a[i]`のxorを取るだけと分かる
