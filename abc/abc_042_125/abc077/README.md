## D - Small Multiple
* keywords
  - グラフ、01-BFS
* 解法
  - 正整数を頂点とするグラフを考える
    - 辺の重みが、桁和の推移となるよう以下の辺を考える
      - 1の位が9以外の時、`n`から`n+1`にコスト1の辺をはる
      - `n`から`10n`にコスト0の辺をはる
  - 1から`K`の倍数への最短経路問題を考えれば良い
    - このままだと頂点数が無限
  - ここで、整数を`mod K`で同一視する
  - その上で上記のようなグラフを作り、最短経路問題を解いてみる
    - `dist[i]`は`mod K`の値が`i`の数値たちの桁和の最小値となる
* comment
  - グラフを考えて、それを`mod K`で考えるとか言うのが難しすぎ
