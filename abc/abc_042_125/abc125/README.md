## C - GCD on Blackboard (300点)
* 複数個のGCDを効率よく求める問題
  - 「好きな数字を別の好きな数に一つ書き換える」＝「それを無視する」
  - aiを無視してGCD = GCD(GCD(a[:i]), GCD[i+1:])
* セグ木でも解ける
  - GCDの単位元は0


## D - Flipping Signs (400点)
負の数が全部で偶数なら頑張ると全部のマイナスを消せる。 \
負の数が全部で奇数なら頑張るとどれか好きな一つまでマイナスを減らせる。 \
これに気づけばソートして計算するだけ。

DPでもやれるらしい。
