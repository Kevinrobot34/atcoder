## D - 三角形の分類
* keywords
  - 幾何、偏角ソート
* 解法
  - 各三角形の定義を考える
    - 鈍角三角形
      - **ある一つ**の角が90度を超える
    - 直角三角形
      - **ある一つ**の角が90度を超える
    - 鋭角三角形
      - **全て**の角が90度未満
      - 鈍角三角形でも直角三角形でもない
  - 鈍角・直角三角形については一つの頂点のみで判定が可能
