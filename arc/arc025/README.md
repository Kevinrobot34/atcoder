## B - チョコレート
* keywords
  - 二次元累積和
* 解法１
  - 白と黒それぞれ用の二次元累積和を用意する
  - `O(H^2W^2)`
* 解法２
  - 二次元累積和だが、白か黒片方を-1倍してからお行う
    - 累積和が一つで済んで嬉しい


# C - ウサギとカメ
* keywords
  - Dijkstra、二分探索
* 解法
  - そのままやろうとするとWarshall-Floydしたくなるが、`N<=2500`より`O(N^3)`は無理
  - `M<=3000`なので、目的地を始点にDijkstra(`O(MlogN)`)を`N`回なら行けそう
  - Dikstraの結果の最短経路に関するリストをソートして二分探索
    - 同じ頂点を使えないことを忘れずに
