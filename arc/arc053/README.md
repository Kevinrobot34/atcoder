## B - 回文分割
* keyword
  - 回文
* 解法
  - 各アルファベットの出現回数の偶奇を見る
  - 奇数個出現するアルファベットの種類は分割しないといけない
    - まず奇数個出現するアルファベットを1つずつ回文の中心におく
    - そうすると残りは全て偶数個になるので、これらを均等に割り振る
