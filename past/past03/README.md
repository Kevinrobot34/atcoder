# PAST03
* https://atcoder.jp/contests/past202005-open

## C - 等比数列
* 解法
  - 等比数列なので、愚直に `10**9` を超えない範囲でシミュレーションすれば良い
    - `O(log 10**9)`


## D - 電光掲示板
* 解法
  - 面倒だけどやるだけ


## E - スプリンクラー
* 解法
  - 隣接リストでグラフを表現し、愚直にクエリをこなすだけ
  - `O(M+NQ)`


## F - 回文行列
* 解法
  - `O(N^3)`とならないようにsetなどを使いながら実装するだけ

## G - グリッド金移動
* keyword
  - BFS、コーナーケース
* 解法
  - 基本的にはBFSするだけ
  - `-200 <= x <= 200 and -200 <= y <= 200`の正方形が基本的な対象だが、この外側の領域を通るのが最適なこともあることに注意
